## Import from external file
```html
<link rel="stylesheet" href="style.css" />
```

## Inline
```html
<div style="color: red">some text</div>
```

## In document

```html
<style>
  .classname {
	color: red;
  }
</style>
```

## What to use
- Prefer external file
- Only use inline when hacking/testing
