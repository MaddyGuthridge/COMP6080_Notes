### `color`
Colour of the content (name of color or hex)
```css
.my-text {
  color: red;
}
```

### `font-weight`
Weight of font
- `bold`
- `thin`
- etc

### `font-family`
Family of font

### `font-size`
- Emmets - `1em` = standard font size
- Percent - `100%` = standard font size
- Number of points (eg `12`)
