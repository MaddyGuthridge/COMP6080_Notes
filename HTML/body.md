- `<p>` paragraph
- `<br>` line break
- `<b>` bold
- `<i>` italics
- `<u>` underline
- `<table>` take a guess
	- `<tr>` table row
	- `<th>` table header cell
	- `<td>` table cell
- `<ul>` unordered list
	- `<li>` list item
- `<ol>` ordered list
- `<span>` horizontal sections (see [[span]])
- `<div>` vertical sections (see [[div]])
- `<img>` image
- `<a>` [[anchor]] (used for links)
- `<style>` [[CSS]] within the HTML document (see [[style]])
